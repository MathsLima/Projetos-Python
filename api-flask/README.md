### API com Flask

API que acessa as informações de uma base de dados e disponibiliza a quantidade de itens vendidos desta base de dados.

Foi utilizado bibliotecas pandas e flask e base de dados xlsx.

## Como rodar o projeto

- Download da base de dados

- Instalar biblioteca Pandas
```
pip install pandas
```
- Instalar biblioteca Flask
```
pip install flask
```

Para que fosse possivel rodar a requisiçao desta API, o codigo foi inserido na palataforma Replit (https://replit.com/) onde é feito o deploy.
